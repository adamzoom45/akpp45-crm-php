# АКПП Курган - WordPress Theme

Профессиональная тема WordPress для сайта по ремонту АКПП.

## Установка

1. Скачайте ZIP-архив темы
2. Войдите в админ-панель WordPress
3. Перейдите в **Внешний вид → Темы → Добавить новую**
4. Нажмите **Загрузить тему**
5. Выберите скачанный файл `akpp-kurgan.zip`
6. Нажмите **Установить**
7. После установки нажмите **Активировать**

## Настройка

### Контактная информация

Перейдите в **Внешний вид → Настроить → Контактная информация**:
- Телефон 1
- Телефон 2
- Telegram
- Адрес

### Главный экран (Hero)

Перейдите в **Внешний вид → Настроить → Главный экран (Hero)**:
- Заголовок
- Описание

### Меню

Перейдите в **Внешний вид → Меню**:
- Создайте меню для позиции "Главное меню"
- Создайте меню для позиции "Меню в подвале"

### Виджеты

Перейдите в **Внешний вид → Виджеты**:
- Подвал - Колонка 1
- Подвал - Колонка 2
- Подвал - Колонка 3

## Custom Post Types

Тема включает следующие типы записей:
- **Услуги** - для добавления услуг
- **Прайс-лист** - для управления ценами

## Форма заявки

Форма заявки отправляет письма на email администратора WordPress.

## Требования

- WordPress 5.0+
- PHP 7.4+

## Поддержка

По вопросам обращайтесь к разработчику темы.
